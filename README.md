# Ali Nazir Website

A static GitHub Pages website with:

- a polished landing page
- a separate blog archive
- a first long-form post on the shift from MDCAT goals to computer engineering

Key files:

- `index.html` for the homepage
- `blog.html` for the weekly post archive
- `posts/` for individual article pages
- `styles.css` for the shared visual system
- `main.js` for blog card rendering, reveal animations, and mobile navigation

The site is intentionally build-free so it can be hosted directly from the repository root on GitHub Pages.
